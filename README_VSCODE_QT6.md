# Configuração do Qt Creator 18.0.2 no VS Code\n\n## Ambiente Detectado\n- Qt: 6.10.2 (MinGW 64-bit)\n- Compilador: MinGW 13.1.0 (g++ 13.1.0)\n- Local: C:\\Qt\\6.10.2\\mingw_64\n\n## Configurações Aplicadas\n\n### 1. Arquivo do Projeto (.pro)\n- Atualizado para C++17\n- Removidas entradas duplicadas\n- Estrutura otimizada para Qt 6\n\n### 2. Configurações do VS Code (.vscode/)\n- **c_cpp_properties.json**: IntelliSense para Qt 6\n- **settings.json**: Associações de arquivo\n- **tasks.json**: Tarefas de build (qmake + mingw32-make)\n- **launch.json**: Debug com GDB\n\n## Como Usar\n\n### Build e Compilação\n1. Abra o comando rápido (Ctrl+Shift+P)\n2. Digite \"Tasks: Run Build Task\"\n3. Selecione \"Qt6 Build (qmake)\" primeiro\n4. Depois execute \"Qt6 Make (compile)\"\n\nOu execute diretamente:\n- **Atalho**: Ctrl+Shift+B (executa a tarefa padrão)\n\n### Executar a Aplicação\n1. Abra o arquivo principal (main.cpp ou mainwindow.cpp)\n2. Vá para a aba \"Run and Debug\" (Ctrl+Shift+D)\n3. Clique no botão \"Start Debugging\" (F5)\n4. Selecione \"Debug Qt App\"\n\n### Limpar o Build\n1. Ctrl+Shift+P -> \"Tasks: Run Task\"\n2. Selecione \"Qt6 Clean\"\n\n## Próximos Passos\n\n### Extensões Recomendadas\n- **C/C++ (Microsoft)**: Já deve ter\n- **Qt Configuration (Filip Krikava)**: Auxilia na configuração\n- **CMake Tools**: Se quiser usar CMake em vez de qmake\n\n### Se algo não funcionar\n1. Verifique se o Qt Creator 18.0.2 está instalado corretamente\n2. Valide que o PATH está correto para mingw32-make\n3. Execute `qmake --version` no terminal para verificar\n\n## Executável Gerado\n- Caminho: C:\\ATM_Project2\\release\\ATM_Project.exe\n- Configuração: Release (otimizado)\n\n## Estrutura de Arquivos do Projeto\n```\nATM_Project2/\n├── .vscode/\n│   ├── c_cpp_properties.json     (IntelliSense)\n│   ├── settings.json             (Configurações)\n│   ├── tasks.json                (Tarefas de build)\n│   └── launch.json               (Debug)\n├── account.cpp / account.h\n├── bankdatabase.cpp / bankdatabase.h\n├── mainwindow.cpp / mainwindow.h\n├── main.cpp\n├── ATM_Project2.pro              (Arquivo de projeto)\n└── Makefile                      (Gerado pelo qmake)\n```\n\n---\n\n**Nota**: Todas as ferramentas estão configuradas com caminhos absolutos para garantir funcionamento mesmo que o PATH não esteja definido corretamente.\n
